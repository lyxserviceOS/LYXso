# Multi-lokasjon & Ressursstyring

## Hvordan bruke

### Steg 1: Legg til lokasjoner
1. Gå til **Innstillinger → Lokasjoner**
2. Klikk "Ny lokasjon"
3. Fyll inn:
   - Navn (f.eks. "Hovedverksted Oslo")
   - Adresse
   - Postnummer og sted
   - Telefon og e-post (valgfritt)
4. Marker én lokasjon som "Primær lokasjon"
5. Klikk "Lagre"

### Steg 2: Legg til ressurser
1. Gå til **Innstillinger → Ressurser**
2. Klikk "Ny ressurs"
3. Fyll inn:
   - Navn (f.eks. "Løftebukk 1")
   - Beskrivelse (valgfritt)
   - Velg lokasjon
   - Velg type (bås, løftebukk, rom, utstyr, annet)
   - Sett kapasitet (hvor mange kan bruke samtidig)
   - Velg farge for visuell identifikasjon
4. Klikk "Lagre"

## Ressurstyper

- **Bås** - Vaskehall, poleringsbås osv.
- **Løftebukk** - For å løfte kjøretøy
- **Rom** - PPF-rom, coating-rom osv.
- **Utstyr** - Spesialutstyr som kan reserveres
- **Annet** - Andre typer ressurser

## Kapasitetsstyring

Hver ressurs har en kapasitet som bestemmer hvor mange bookinger som kan bruke den samtidig:

- **1** - Kun én booking om gangen (standard)
- **2-10** - Flere bookinger kan bruke ressursen samtidig

Systemet sjekker automatisk tilgjengelighet når noen prøver å booke.

## Tips

- Bruk farger for å skille mellom ressurstyper visuelt
- Gi ressurser beskrivende navn (f.eks. "Løftebukk 1 (høyre)", "Vaskehall A")
- Marker kun én lokasjon som primær
- Ressurser kan ikke slettes hvis de har fremtidige bookinger

## Fremtidige funksjoner

- Filter bookinger per lokasjon i kalender
- Filter per ressurs
- Kapasitets-indikator i kalender
- Åpningstider-editor for lokasjoner
- Statistikk over ressurs-utnyttelse
